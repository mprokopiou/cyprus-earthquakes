Live Cyprus Earthquake Map

This map displays recent earthquakes around Cyprus based on live data from the European-Mediterranean Seismological Centre (EMSC). Each marker represents an earthquake, color-coded by how recent it was: green for the last 24 hours, orange for the last 3 days, and red for older events.

Hover or click on a point to see detailed information, including magnitude, location, and the local Cyprus time of the event.
